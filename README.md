# quiz-app-FE# Quiz App: Naija Trivia 🇳🇬

This is a simple JavaScript-based quiz app where users can test their knowledge of Nigeria, African culture, and basic frontend development concepts. It includes multiple-choice questions covering topics like Nigerian food, slang, cities, and web development basics.

---

## Features

* Presents one multiple-choice question at a time
* Tracks the user's score based on correct answers
* Displays the current question number (e.g., "Question 2 of 12")
* Includes a countdown timer for each question
* Highlights correct and incorrect answers
* Shows the final score at the end of the quiz
* Allows the user to restart the quiz

---

## Technologies Used

JavaScript (main logic)
HTML (basic structure)
CSS (simple styling)

---

## How to Run

### Clone or Download the Code

You can clone the repository or download the files to your computer.

### Open the Project

Open the `index.html` file in your browser.

### Start the Quiz

Once the page loads, the quiz will start automatically.
Select an answer for each question before the timer runs out.
At the end, your score will be displayed and you can restart the quiz.

---

## How It Works

**Question Data:**
Questions are stored in an array containing the question text, answer options, and the correct answer.

**Displaying Questions:**
JavaScript dynamically updates the page to show one question and its options at a time.

**Answer Selection:**
When an answer is selected, it is checked against the correct answer and the score is updated.

**Timer:**
Each question has a countdown timer. If time runs out, the question is automatically skipped.

**Progress Tracking:**
The app shows which question you are currently on (e.g., "Question 5 of 12").

**End of Quiz:**
After all questions are answered, the final score is displayed along with a restart button.

# Quiz App

This is a basic quiz application built with HTML, CSS, and JavaScript as part of my frontend learning.

The app displays one question at a time with multiple-choice answers. When a user selects an option, the app checks if it’s correct and updates the score. It also shows the current question number and includes a timer for each question.

At the end of the quiz, the final score is displayed along with a restart button that allows the user to take the quiz again.

This project helped me practice working with arrays, functions, event listeners, and DOM manipulation in JavaScript.
